Including the rumble plugin

						//you'll need to have jQuery included
						<script type="text/javascript" charset="utf-8" src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js"></script>	
						
						//then include the version of the plugin you want, I chose the minified
						<script type="text/javascript" charset="utf-8" src="/js/jquery.rumble.min.js"></script>

The Most Basic Rumble

I call this the most basic example because it uses all the defaults and simply requires instantiating the rumble plugin. 

						$('#input1').focus(function(){
							$(this).rumble();
						});

A Basic Vertical Rumble

This example is hardly more complicated than the basic one, it's simply going to rumble vertically instead of horizontally. This is accomplished by passing the 'axis' parameter. 

						$('#input2').focus(function(){
							$(this).rumble({ axis:'y' });
						});

A More Customized Rumble

Click me to watch me rumble!

This time let's set a few more parameters to make the rumble a little more unique. Let's also use a div instead of an input. 

						$('#example3').click(function(){
							$(this).rumble({
								distance: 15,
								shakes: 12,
								speed: 130,
								shrinkDistance: true
							});
						});

Rumble with Callback

Click me!
Waiting...

This example adds in a callback function for when the rumble has completed. In this case, we rumble something else, but you could do whatever you want of course. 

						$('#example4a').click(function(){
							$(this).rumble({
								shakes: 8,
								onComplete: function(){
									$('#example4a').text('Done!');

									$('#example4b').rumble({
										shakes: 6
									}).text('My turn!');
								}
							});
						});
